FIX: HISTORICO_CHAT.kt - Erros de Compilacao
========================================

ERROS ENCONTRADOS:
1. Only 'const val' can be used in constant expressions (linhas 14, 15)
2. Annotation argument must be a compile-time constant (linha 16)
3. Unresolved reference 'UsuarioPerfil' (linha 16)
4. Package directive does not match the file location (linha 1)

========================================
SOLUCAO
========================================

ERRO 1 e 2: const val em Table name
=======================================
PROBLEMA:
Linhas 14-15 usam 'var' quando deveriam usar 'const val'
Annotations no Room exigem compile-time constants

FIX:
Mude estas linhas:
```kotlin
var tableName = "historico_chat"    // ERRADO
var columnId = "id"                  // ERRADO
```

Para isto:
```kotlin
const val TABLE_NAME = "historico_chat"
const val COLUMN_ID = "id"
```

Depois atualize a annotation:
```kotlin
@Entity(tableName = TABLE_NAME)  // Correto
```

========================================
ERRO 3: Unresolved reference 'UsuarioPerfil'
=======================================
PROBLEMA:
A classe UsuarioPerfil nao existe ou nao foi importada

OPCAO 1: Se a classe deve existir:
```kotlin
import com.dutra.agente.data.banco.entidades.UsuarioPerfil
```

OPCAO 2: Se nao precisa da referencia:

OPCAO 3: Se precisa, crie a classe:
Arquivo: app/src/main/java/com/dutra/agente/data/banco/entidades/UsuarioPerfil.kt

========================================
ERRO 4: Package directive does not match
=======================================
PROBLEMA:
A declaracao 'package' na linha 1 nao corresponde ao caminho do arquivo

ARQUIVO: HistoricoChat.kt
CAMINHO: app/src/main/java/com/dutra/agente/data/banco/entidades/

DEVE SER:
```kotlin
package com.dutra.agente.data.banco.entidades
```

NAO:
```kotlin
package com.dutra.agente.data.banco  // ERRADO
```
